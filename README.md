# 👥 HR Analytics Dashboard – Power BI

An interactive HR Analytics dashboard built using Power BI to analyze employee attrition, workforce trends, job satisfaction, overtime impact, and overall employee performance using the IBM HR Analytics dataset.

---

# 🚀 Project Overview

This project transforms raw HR employee data into meaningful business insights through interactive dashboards and visual analytics.

The dashboard helps organizations understand:
- Why employees leave
- Which departments face higher attrition
- The impact of overtime on employee retention
- Workforce demographics and trends
- Employee satisfaction patterns

---

# ✨ Dashboard Features

## 📌 KPI Metrics
Track important HR metrics including:
- 👨‍💼 Total Employees
- 📉 Attrition Count
- 📊 Attrition Rate
- 💰 Average Salary
- 🎂 Average Employee Age

---

## 📍 HR Analysis
Analyze workforce trends across:
- Departments
- Gender
- Age Groups
- Job Roles
- Overtime Status

---

## 📅 Employee Insights
- Attrition by Department
- Overtime vs Attrition
- Attrition by Age Group
- Job Satisfaction Analysis
- Workforce Demographics

---

## 🎛️ Interactive Dashboard
Users can dynamically explore data using:
- Department filters
- Gender slicers
- Overtime filters
- Interactive visual exploration

# 📈 Key Business Insights

✅ Employees working overtime showed higher attrition  
✅ Younger employees experienced higher turnover rates  
✅ Sales department had the highest attrition  
✅ Lower salary employees were more likely to leave  
✅ Employee satisfaction influenced retention trends
